Esta pasta e subpastas contém dados e scripts para a ferramenta de Dissecação Horizontal. 
- Para executar a ferramenta abra uma interface do ArcCatalog e acesse a toolbox "Dissecação Horizontal.tbx" e clique na ferramenta "Dissecação Horizontal". 
- Para informações sobre o funcionamento da ferramenta e dados espaciais necessários consulte a subpasta "Doc" e a aba lateral da ferramenta.
- O arquivo "Dissecação Horizontal - Exemplo.mxd" apresenta exemplo de dados espaciais que são necessários para executar a ferramenta e o resultado alcançados com o seu processamento.
- A ferramenta "Reclassificar Dissecação Horizontal" deve ser utilizada apenas em um segundo momento caso seja necessário uma nova classificação.

Avisos:
Esta ferramenta foi desenvolvida e testada na versão 10.2 do software ArcGIS.
Todos os dados vetoriais de entrada devem estar localizados dentro de um banco de dados. Não utilizar arquivos no formato Shapefile.
Para referenciar esta ferramenta utilize o artigo: FERREIRA, M. V.; TINÓS, T. M.; PINTON, L. D. G.; CUNHA, C. M. L. A dissecação horizontal como parâmetro morfométrico para avaliação do relevo: proposta de técnica digital automática. Revista Brasileira de Geomorfologia, v. 15, n. 4, p. 585-600, 2014.
